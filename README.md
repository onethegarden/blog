# 블로그 프로젝트


## 🔧 사용된 기술 
- frontend : React.js
- backend : Koa.js


## 🚀 시작하기

### Install
#### npm 설치
> `blog frontend` 디렉토리에서 실행
```
cd blog-frontend
npm install
```

> `blog backend` 디렉토리에서 실행
```
cd blog-backend
npm install
```

### Usage

#### koa server 구동

> `blog backend` 디렉토리에서 실행
재기동 없이 : start:dev
재기동 필요 : start
```
yarn start:dev
```
#### react 구동
> `blog frontend` 디렉토리에서 실행
```
yarn start
```
